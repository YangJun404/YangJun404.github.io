我是 YaJn

我的网络名称为 YaJn    由真实名称的拼音删减而来。由于家庭原因，我可能会考虑更换一个新的名称。

是一位 CPTSD 患者

家庭原因？不满你所，我在早期就长期遭受着家庭暴力，包括但不限于 语言、肢体。即使现在，这种情况已经不再发生，但是被遗弃感觉在本能脑中似乎根深蒂固，以至于在睡梦中、学习中常常出现 闪回 现象。

爱好 心理学

为了防止自己的言语冒犯他人，或者潜移默化的影响别人。我自打有意识起便对心理学颇有兴趣。为了自我疗愈，我喜欢阅读《不原谅也没关系》等心理学书籍。

同时，因为遗弃感的原因，我对他人不具备信任。因此，我的老师经常被我怀疑。为了让自己相信教育带来的正向结果，我也喜欢 教育心理学。这解释了老师的行为，同时也通过相对客观的结论来解释自己的学习习惯，获得自信。

与服务器维护

不过，面对家庭的暴力、落榜的成绩，我迷恋上了网络世界。这是唯一一个持续给我正反馈，让我发掘自身价值的地方（因为我所持有的错误价值观，我的同学并不理解我的行为举止）。

从 Minecraft 的 单人游戏 到 多人游戏 到 Windows服务器 到 Linux 服务器 再继续往下 Docker , Iptables 甚至 试图学习 Java 语言… 这一段历程让我的计算机技术在同龄人的范围内更高，因此老师常常叫我去修电脑。“你毕业后也不知道找谁去修电脑”–一位老师。

同时是一位高中生，希望相互学习。

现在，至少是一位高一新生。我的中考成绩并没有那么理想，没有考上心仪的高中。但是我一直相信，我们从来不是被定义的。

在之后，我会在空余时间用书籍找回残缺的人格。希望大家也能与我沟通，交流不同意见。

小结

请重新阅读一次副标题，这就是我的大概人格。

谢谢读者用空余时间，了解了可能用处不大的小人物。
